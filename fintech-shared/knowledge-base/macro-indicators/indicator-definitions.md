## 各指标详细定义

### 1. 移动支付交易规模
- **定义：** 一定时期内，通过移动设备（智能手机、平板）完成的支付交易总金额
- **统计口径：** 包括NFC支付、扫码支付、P2P转账、移动网关支付
- **区别：** 不包括传统银行卡线下刷卡交易、PC端网上支付
- **单位：** 中国 = 人民币万亿元；香港 = 港元万亿元；其他 = 十亿美元

### 2. 移动支付渗透率
- **定义1（交易口径）：** 移动支付交易金额 / 社会消费品零售总额 × 100%
- **定义2（用户口径）：** 移动支付活跃用户数 / 互联网用户总数 × 100%
- **说明：** 两种口径并存，报告中需明确标注使用哪种定义
- **优先使用：** 交易口径（移动支付交易金额 / 社零）

### 3. 整体消费信贷余额
- **定义：** 金融机构向个人消费者发放的用于消费用途的**贷款余额**（存量，不含住房抵押贷款）
- **包含：** 短期消费贷、信用卡透支、中长期消费贷
- **不包含：** 住房按揭贷款、经营性贷款
- **数据来源：** 各国中央银行统计数据

### 4. 互联网信贷/网贷余额
- **定义：** 由互联网平台参与发放的消费信贷余额存量
- **包含：** 互联网消费金融公司贷款、助贷平台联合贷款、P2P（存量）、BNPL（先买后付）
- **统计口径：** 按照行业报告披露的规模统计

### 5. 电商交易规模
- **定义：** 一定时期内，所有通过互联网完成的零售交易总金额
- **统计范围：** B2C + C2C，包含实物商品和服务类电商
- **不包含：** 企业间B2B批发贸易

### 6. 社零总额（社会消费品零售总额）
- **定义：** 企业通过交易售给个人、社会集团非生产、非经营用的实物商品金额
- **发布频率：** 月度
- **用途：** 反映整体消费市场景气度，是计算渗透率的分母

---

### 7. 政策利率（货币政策利率）
- **定义：** 各经济体中央银行制定的关键政策利率，影响整体融资成本和消费信贷定价
- **包含指标：**
  - 美国：联邦基金目标利率区间（中值）
  - 中国：1年期LPR（贷款市场报价利率） + 七天逆回购利率
  - 欧元区：欧洲央行存款便利利率
  - 英国：英格兰银行基准利率
  - 日本：政策余额利率
  - 香港：基准利率（跟随美联储）
- **战略重要性：** 直接影响消费信贷成本、BNPL定价、支付公司利润率，对Fintech估值影响重大
- **发布频率：** 跟随议息会议（通常每6周一次）

|------|---------|
| `[精确数据]` | 来自央行/统计局官方发布 |
| `[估算数据]` | 来自第三方研究机构/行业报告 |
| `[推测数据]` | 无直接来源，AI推断 |
| `[数据存在差异]` | 不同来源数字矛盾 |
| `[数据缺失]` | 搜索无结果，无法获取 |

## 同比环比计算规则

- **同比（YoY）：** (当期数值 / 去年同期数值 - 1) × 100%
- **环比（MoM/QoQ）：** (当期数值 / 上期数值 - 1) × 100%
- **渗透率变化：** 直接报告百分点变化（如 +2.3 pct）

## 趋势解读模板

生成趋势解读时遵循以下框架：

1. **整体判断：** 行业整体扩张/收缩趋势
2. **关键变化：** 哪个指标增速加快/放缓
3. **驱动因素：** 可能的原因（经济周期/监管/竞争）
4. **战略含义：** 对字节业务进入/扩张的启示
