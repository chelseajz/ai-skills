# 支付与清结算行业基础知识

## 核心定义
支付与清结算是指完成资金从付款方到收款方转移的全过程。包括：
- **支付**：用户发起交易指令
- **清算**：计算各方债权债务
- **结算**：最终完成资金转移

## 细分赛道

| 细分 | 说明 | 核心玩家举例 |
|------|------|--------------|
| 消费者支付 (C2B/C2C) | 个人对商家、个人对个人 | 支付宝、微信支付、PayPal |
| 商户收单 | 商家接收消费者支付 | Stripe、Adyen、银联 |
| 跨境支付 | 跨境汇款、外贸收款 | Wise、Airwallex、PingPong |
| B2B支付 | 企业间付款、供应链支付 | 连连支付、FIS |
| 支付基础设施 | 清算网络、CBDC | 银联、网联、FedNow |
| Agentic Payment | AI代理自主支付决策 | 新兴领域 |

## 产业链与利润分配（中国市场参考）

```
消费者 → 发卡行/钱包 → 清算网络（银联/网联） → 收单机构 → 商家
```

**利润分配（MDR商户折扣率）：**
- 发卡行：约 70%
- 清算机构：约 10%  
- 收单机构：约 20%

**关键指标：**
- TPV (Total Payment Volume)：支付交易总额
- Net Take Rate：净费率（净收入/TPV，通常 0.1%-2%）
- MAU：月活跃支付用户
- 拒付率：争议交易占比

## 核心成功要素

1. **网络效应**：用户越多 → 商家越多 → 用户更多
2. **合规壁垒**：支付牌照极强的进入壁垒
3. **规模效应**：固定成本高，规模越大单位成本越低
4. **场景绑定**：绑定电商/社交场景者得天下

## 常见分析误区（必须避免）

1. **混淆 TPV 和净收入**：TPV 是交易量，不是收入，take rate 很小
2. **支付宝=纯支付**：蚂蚁利润主要来自信贷，不是支付手续费
3. **忽视跨境监管壁垒**：每个国家牌照难度差异极大
4. **低估清算基础设施**：网联/银联看似边缘，实际是战略枢纽

## 当前主要趋势

- 实时支付普及（FedNow、PIX、UPI）
- 数字人民币跨境试点推进
- AI 风控反欺诈应用加速
- 嵌入式支付成为常态
- Agentic Payment 开始产品化探索
