# 消费金融与信贷行业基础知识

## 核心定义
面向个人消费者提供的借贷产品和服务，用于满足消费、应急等资金需求。

## 细分赛道

| 细分 | 说明 | 代表产品 |
|------|------|----------|
| 现金贷/消费贷 | 个人无抵押信用贷款 | 放心借、借呗 |
| 消费分期/BNPL | 购物场景分期 | 月付、花呗、Klarna |
| 数字银行/Neobank | 纯线上银行，提供存款+信贷 | Revolut、Chime、ZA Bank |
| 小微企业信贷 | 面向小微企业经营性贷款 | 网商银行 |
| 助贷模式 | 平台获客+风控，银行出资 | 多数互联网信贷 |

## 核心商业模式与财务结构

**收入来源：**
- 利息收入（主要）：净利差 = 贷款利率 - 资金成本
- 服务费：分期手续费、origination fee
- 交叉销售：信贷用户卖理财/保险

**成本结构：**
- 资金成本：最大成本，受货币政策影响大
- 风险成本：坏账NPL
- 获客成本CAC：互联网平台获客成本持续上升

**关键指标：**
- 贷款余额：规模
- NPL (不良贷款率)：风险水平
- NIM (净息差)：盈利能力
- CAC (获客成本)：获客效率

## 核心成功要素

1. **风控能力**：决定坏账率，直接影响盈利
2. **获客成本**：互联网平台核心壁垒
3. **资金成本**：规模越大、评级越好，成本越低
4. **合规能力**：监管政策变化影响生死

## 中国监管框架要点

- **主管机构**：中国人民银行、国家金融监督管理总局
- **核心法规**：《关于进一步规范商业银行互联网贷款业务的通知》（2020）
- **联合贷款规则**：
  - 出资比例：银行不得低于30%
  - 集中度：单户贷款余额不超过30万
- **牌照要求**：放贷需要牌照，无牌不能经营

## 常见分析误区

1. **只看规模不看坏账**：规模大没用，坏账高会吞噬所有利润
2. **忽视利率环境影响**：加息周期资金成本上升，盈利压缩
3. **助贷模式轻资产误解**：助贷虽然不用出资，但仍需要兜底风险
4. **BNPL混淆支付和信贷**：BNPL本质是信贷，不是支付

## 当前主要趋势

- AI风控技术普及，降低坏账率
- 监管全球趋严，尤其BNPL
- 东南亚等新兴市场增速快于欧美
- 平台流量变现重要方式
