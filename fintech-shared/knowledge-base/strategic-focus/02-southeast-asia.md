# 战略重点区域：东南亚（Southeast Asia）

## 战略定位
- **字节优先级**：⭐ 最高优先级
- **原因**：TikTok 已拥有庞大用户基础和快速增长的电商业务，支付+信贷是自然延伸
- **主要国家**：印尼、越南、新加坡、泰国、菲律宾

## 市场整体概况

| 指标 | 数据 | 说明 |
|------|------|------|
| 总人口 | ~6.8亿 | 庞大年轻人口基数 |
| 互联网渗透率 | ~73% | 持续增长中 |
| 电商GMV | ~$2000亿+ | 快速增长，TikTok Shop 扩张中 |

## 竞争格局（按赛道）

### 支付
- **本土玩家**：GoPay (印尼/Gojek), ShopeePay (Sea), DANA (印尼), OVO (印尼), MoMo (越南), VNPay (越南), GrabPay (区域)
- **SEA/Grab**：两大生态集团都已拿到支付+信贷牌照
- **外资**：Airwallex, Stripe (正在进入)

### 信贷/BNPL
- **Akulaku**：印尼最大本土分期平台
- **SeaMoney**：ShopeePay + 信贷，依托Shopee电商
- **GrabPay Later**：依托Grab生态
- **Tala**：全球小额信贷玩家

### 数字银行
- **SeaBank**：印尼，Sea集团旗下，快速扩张
- **GoTo Bank**：印尼，GojekTokopedia

## 对字节战略意义
1. **TikTok Shop 电商快速增长** → 支付+分期是提升转化率刚需
2. **用户基础已具备** → 获成本低于从零开始
3. **路径和中国一致** → "电商流量 → 支付 → 信贷"可复制中国成功模式
4. **竞争格局**：Sea 是直接竞争对手（模式一样：电商+支付+信贷）
