评分方法

维度 权重 评分规则
信源可信度 25分 一级=25分 / 二级=20分 / 三级=15分 / 四级=10分
时间紧迫性 10分 本月=10分 / 上月=5分 / 更早=3分
信息独特性 25分 模型判断：独家=25 / 增量=20 / 已知=10
战略相关性 25分 模型判断：直接相关=25 / 间接=15 / 弱相关=5
影响持续性 10分 模型判断：长期=10 / 中期=7 / 短期=3
市场热度 5分 模型判断：高=5 / 中=3 / 低=1

工作流程

  1. Phase 1 - 一级信源抓取：直接从监管官网（HKMA、Fed、ECB等）抓取新闻
  2. Phase 2 - 二级信源搜索：103个查询覆盖所有核心公司，官网优先策略
  3. Phase 3 - 去重与完整性检查：基于URL去重，检查并购/新产品等关键事件是否遗漏
  4. Phase 4 - 六维评分：信源可信度+时间+独特性+相关性+持续性+热度
  5. Phase 5 - 深度分析：对≥80分事件进行信号/背景/影响/缺口/行动五维分析
  6. Phase 6 - 报告生成：Markdown + 交互式 HTML（4主题）
  7. Phase 7 - 质量校验：自动化脚本验证占位符/章节/链接
