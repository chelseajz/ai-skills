## 宏观影响分析框架

货币政策利率对 Fintech 行业的传导路径：

1. **信贷类业务：**
   - 利率下降 → 资金成本下降 → 消费信贷投放扩张 → 利润率提升
   - 利率下降 → BNPL 融资成本下降 → 费率可降 → 竞争力提升

2. **支付类业务：**
   - 利率下降 → 消费信贷扩张 → 交易金额增加 → 手续费收入增长
   - 支付公司备付金利息收入下降，但这部分收入占比普遍不高

3. **估值影响：**
   - 利率下降 → 折现率下降 → 成长型 Fintech 公司估值提升
   - 对高杠杆商业模式利好更明显

4. **美国降息周期特殊影响：**
   - 美元流动性宽松 → 新兴市场资本流入 → 东南亚/拉美 Fintech 融资环境改善
   - 美元走弱 → 大宗商品价格企稳 → 新兴市场消费能力边际改善
