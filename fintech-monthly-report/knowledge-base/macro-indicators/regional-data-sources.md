# 各区域宏观指标数据源清单

## 搜索优先级规则

1. **一级信源优先：** 各国中央银行官网 → 国家统计局 → 金融监管当局
2. **二级信源：** 行业协会报告 → 权威财经媒体
3. **三级信源：** 咨询公司研究报告 → 上市公司财报披露
4. 必须标注信源等级和来源链接

## 搜索查询模板

### 中文搜索模板

```
"{区域} {指标名称} {YYYY}年{MM}月 数据"
"{国家} 移动支付交易规模 {YYYY} 季度"
"{国家} 消费信贷余额 {YYYY}年 最新"

示例:
"中国 消费信贷余额 2026年3月"
"香港 移动支付交易规模 2026 一季度"
"印尼 电商交易规模 2025 年度"
```

### 英文搜索模板

```
"{country} {indicator name} {Month} {Year} data"
"{country} {indicator} {YYYY} Q{quarter} statistics"

示例:
"US consumer credit outstanding March 2026"
"Hong Kong mobile payment transaction volume 2026 Q1"
"Indonesia e-commerce turnover 2025"
```

## 数据采集工作流

每次生成月报前，按以下步骤执行：

1. **加载模板数据：** 读取 `knowledge-base/macro-indicators/templates/` 下对应区域CSV文件（历史参考数据）
2. **检查目标月份数据：** 如果目标月份数据已存在于 `~/.claude/fintech-reports/data/macro-indicators/`，直接使用
3. **确定缺失数据：** 列出需要搜索更新的区域和指标
4. **按优先级搜索：** 中国 → 美国 → 欧洲 → 香港 → 日本 → 韩国 → 东南亚 → 墨西哥 → 巴西 → 澳大利亚 → 非洲
5. **提取数据：** 从搜索结果中提取最新数值，记录来源和URL
6. **更新CSV：** 追加新数据行到对应CSV文件（保存到 `~/.claude/fintech-reports/data/macro-indicators/`）
7. **计算变化：** 自动计算同比和环比变化
8. **生成报告：** 将完整表格插入月报第二章"全球宏观指标速览"

## 信源访问规则

遵循skill现有规则：

- ✅ **安全可直接访问：** 各国央行官网、统计局官网可直接WebFetch
- ⚠️ **受限：** 付费媒体内容只使用公开摘要，标注`[需付费来源，仅获取公开摘要]`
- ❌ **高风险：** 禁止编造数据，无法获取则标注`[数据缺失]`
