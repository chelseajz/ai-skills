# 腾讯金融科技 - 竞争对手深度档案

## 基本信息
- **总部**：中国深圳
- **核心业务**：微信支付、财付通、微粒贷、理财通
- **字节对标**：支付

## 核心业务结构

| 业务 | 说明 |
|------|------|
| 微信支付 | 社交+支付，网络效应极强 |
| 微粒贷 | 微众银行个人信贷 |
| 理财通 | 财富管理 |

## 核心护城河
1. **社交网络效应**：微信社交关系链带来极致粘性，用户日常付款首选
2. **牌照齐全**：第三方支付牌照、银行牌照
3. **流量垄断**：占据社交入口，线下支付份额接近支付宝

## 对字节竞争策略
- 微信支付已经占据线下主流，抖音支付主要在抖音电商内部闭环
- 竞合并存：微信封杀抖音链接，但支付互联互通后也接受抖音付款
- 抖音电商闭环对微信支付依赖降低，字节支付成长空间在自有场景
