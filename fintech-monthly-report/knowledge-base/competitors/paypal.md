# PayPal - 竞争对手深度档案

## 基本信息
- **总部**：美国加州圣何塞
- **核心业务**：数字钱包、跨境支付、Pay Later BNPL、加密支付
- **市值**：~$800亿
- **字节对标**：跨境支付

## 核心业务结构

| 业务 | 说明 |
|------|------|
| 个人钱包/支付 | C2C/B2C支付 |
| 跨境支付 | 商家跨境收款 |
| Pay Later | BNPL先买后付 |
| 加密支付 | 买卖/持有加密货币 |

## 核心护城河
1. **网络效应**：全球品牌认知高，卖家用户都用
2. **跨境基础设施**：多年 build 起来的网络，新进入者难替代
3. **品牌**：全球最早数字支付品牌

## 战略最新动向
- 加密支付持续投入
- AI 升级风控和用户体验
- 新兴市场（拉美/东南亚）扩张
